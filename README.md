# V2RayZero
V2RayZERO是在  https://github.com/kxjhlele/v2ray-taskbar  基础的之上添加了UI操作


V2Ray的官方主页
https://www.v2ray.com/


需要有官方的内核文件在同一个目录下，以及需要config.json文件在同一个目录下

Release 文件在 V2RayZero/V2RayZero/bin/Debug/  这个路径下面


确保客户端配置文件中存在

"streamSettings": {
	"network": "tcp
  }
  

以及users中存在

"security": "aes-128-cfb",
